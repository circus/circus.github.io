Taxonomy of Inconsistency Patterns in Multi-View Modelling
Category C7: Temporal skew

Views are individually plausible but inconsistent because they reflect different points in evolution, propagation, or branching history.

C7 captures drift caused by asynchronous evolution. Examples include code customised after generation while the source model remains unchanged, models staying stale for long periods while implementations evolve, and parallel versions producing conflicting states for the same method
